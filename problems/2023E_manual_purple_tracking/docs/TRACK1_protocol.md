# TRACK1 protocol

MaixCAM -> ZET6，严格 12 字段：

```text
$MV,TRACK1,OK,TARGET_CX,TARGET_CY,AIM_CX,AIM_CY,EX,EY,SCORE,FPS,STATUS#
```

`OK` 只能为 0/1；`EX=TARGET_CX-AIM_CX`，`EY=TARGET_CY-AIM_CY`；状态为 `TRACKING/AIMED/NO_TARGET/LOST/ERROR`。

ZET6 -> PC debug：

```text
$DBG,TRACK1,EX=-20,EY=15,PAN=-300,TILT=220,VALID=1,STATUS=TRACKING,STATE=TRACKING#
```

PAN/TILT 为整数毫单位建议值，不是角度或 PWM。未来抽象格式可为 `$GM,CMD,PAN=-300,TILT=220,MODE=TRACK#`，但在底层协议审查完成前只允许生成字符串，不允许发送。
