# STM32F103C8T6 云台电机板资料审查

## 找到的资料

原始资料位于仓库工作区父目录 `DCC-100双路步进电机驱动模块资料/`：

- `DCC-100-stm32f103c8t6/DCC-100-stm32f103c8t6/`：CubeMX `.ioc`、Keil 工程、`Core/Inc/motor.h`、`Core/Src/motor.c` 和 `main.c`。
- `DCC-100-stm32f103c8t6_2026_04_29更新/DCC-100-stm32f103c8t6/`：更新版同类工程。
- `DCC-100v1v2说明书.pdf`、`DCC-100v3说明书-2026-05-24.pdf`：DCC-100 双路步进驱动接口资料。
- MSPM0G3507 的基础步进、速度/角度和云台按键开环示例，只作交叉参考，不是 C8T6 协议。

## 板卡和固件性质

两版 `.ioc` 均明确目标 MCU 为 `STM32F103C8T6`，因此 C8T6 板有自己的 MCU 固件。当前 CubeMX 只启用了 RCC、SYS、GPIO、TIM2/NVIC；UART、USART、I2C 模块没有启用。

当前控制对象是两路步进电机，不是直流电机、舵机或无刷云台电机。C8T6 通过 GPIO 产生脉冲并控制 DCC-100：

- 左右轴：PB3 脉冲、PB4 DIR、PB5 DCY、PB6 SLP、PB7 RST。
- 上下轴：PA8 脉冲、PB15 DIR、PB14 DCY、PB13 SLP、PB12 RST。

更新版局部引脚可能有差异，接线前必须以实际 PCB 和最终固件再核对，不能只抄旧版注释。

## 已有控制能力

源码提供 `Motor_Init`、`Motor_SetSpeed`、`Motor_Start`、`Motor_Stop`、`Motor_SetDirection`、`Motor_Turn_Angle`，其中 motor 0 表示左右、motor 1 表示上下。速度和角度通过 TIM2 中断的软件计数/脉冲翻转实现。

这属于开环步进控制。未找到编码器输入、PID、速度闭环、位置闭环、回零、机械限位开关处理、堵转检测、过流保护、看门狗通信超时或急停状态机。DCC-100 自身提供 STEP/PWM、DIR、DCY、SLP、RST 和 GND，不是带上层协议的智能控制器。

## 通信和协议

当前 C8T6 固件没有 UART/USART、I2C 或命令输入，也没有 `$GM,CMD` 或其他通信协议。原 `main.c` 会在无限循环中主动执行两轴 30 度往返测试，上电即可能产生真实电机脉冲，因此当前固件不能用于“只接线不动作”的阶段，也不得直接烧录后连接电机。

## 能否作为底层云台控制板

C8T6 的 MCU、定时器和两轴函数表明它具备成为底层步进云台控制板的基础，但当前固件不具备安全从机协议和闭环/限位保护，尚不能直接接入 ZET6 实时追踪链路。要成为底层板，后续至少需要单独审查并实现：默认禁能、UART 命令 parser、命令超时归零、限速/加速度、软硬限位、回零、急停和故障回报。本任务不修改 C8T6 固件。

## 推荐 ZET6 接入方式

推荐未来使用 ZET6 的 USART2 或 USART3 向 C8T6 UART 发送经过确认的定长 ASCII/二进制协议；USART1 继续保留给现有 PC debug bridge。选择 UART 是架构建议，不代表当前已有协议。I2C 没有现成实现；直接由 ZET6 输出 STEP/PWM 会绕过 C8T6，不符合当前分层。

## 仍不确定

- C8T6 实物对应旧版还是 2026-04-29 更新版，PCB 引脚是否完全一致。
- C8T6 板自身输入电压、稳压方式、串口引脚引出位置及逻辑电平。
- 两轴步进电机型号、额定电压/电流、相序、减速比和机械行程。
- DCC-100 是 V1/V2 还是 V3，LC/HC 电流设置及 DCY/SLP/RST 默认状态。
- 是否安装限位开关、编码器、机械挡块或独立急停。
- C8T6 与 ZET6 是否共电源、是否需要电平转换/隔离，以及电机噪声对逻辑电源的影响。

## 接线前必须确认

1. ZET6 与 C8T6 的供电电压、UART 逻辑电平，禁止把 5V TX 直接送入不耐受引脚。
2. TX/RX 交叉方向：ZET6 TX -> C8T6 RX；若需回传，再接 C8T6 TX -> ZET6 RX。
3. 非隔离 UART 必须共地；若使用隔离器，则按隔离器电源域接线。
4. 电机 11-26V 电源与 MCU 逻辑电源的地、隔离、保险和断电顺序。
5. 电机未接时 C8T6 是否会安全启动；当前原始固件答案是否定的，因为 main 循环主动运行电机函数。
6. 限位、回零、急停/总电源断开方式必须在空载测试前可用并由人员触达。
