# 2023E 单云台 subset replay

这是按当前实物能力建立的独立题目目录，不等同于完整 2023E。

当前硬件只有一个二维云台，云台上安装 MaixCAM/摄像头和一个安全可见、低功率光点发射器，STM32ZET6 作为实验主控；没有第二套独立云台。当前阶段不接真实高功率激光、不启用 405nm、不输出 PWM，也不控制真实云台、舵机、小车或机械臂。

可验证的子集：回到屏幕中心、追踪目标点、屏幕正方形路径、A4 黑胶带矩形及其旋转路径，并记录收敛时间、误差和抖动。日志可由本地脚本分析，后续 API 只能给人工审核的建议；YOLO 仅作为 A4/屏幕检测后备。

不能声称完成：独立红色目标系统、独立绿色追踪系统、两系统无通信，以及完整发挥部分的双光斑追踪。完整合规版本仍需两套相互独立的云台/控制系统。

- `maixvision_project/`：可直接由 MaixVision 打开的自包含工程。
- `mcu_stm32zet6/`：解析、状态机和建议命令的纯计算代码。
- `scripts/`：PC 仿真、日志分析和串口 debug 自动检查。
- `docs/`：协议、架构、安全边界与后续规划。
