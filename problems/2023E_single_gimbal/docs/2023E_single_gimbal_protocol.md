# 2023E SINGLE 文本协议

MaixCAM 到 ZET6：

```text
$MV,SINGLE,OK,SPOT_CX,SPOT_CY,TARGET_CX,TARGET_CY,EX,EY,PATH_ID,STEP_ID,SCORE,FPS,STATUS#
```

严格为 14 个逗号字段。`OK` 只能为 0/1；`EX = SPOT_CX - TARGET_CX`，`EY = SPOT_CY - TARGET_CY`。`PATH_ID` 可为 `CENTER`、`SCREEN_SQUARE`、`A4_RECT`、`A4_ROTATED_RECT` 或 `MANUAL`；状态可为 `TRACKING`、`AIMED`、`NO_SPOT`、`NO_SCREEN`、`NO_RECT`、`LOST`、`ERROR`。

ZET6 到 PC debug 示例：

```text
$DBG,SINGLE,EX=-12,EY=8,PAN=-300,TILT=200,VALID=1,PATH=SCREEN_SQUARE,STEP=12,STATUS=TRACKING,STATE=TRACKING#
```

`PAN/TILT` 是整数毫单位建议值：`-300` 表示 `-0.300`，不得使用 `printf("%f")`。它们不是 PWM、角度或真实执行命令。半帧会缓存，粘包逐帧输出，超长帧和字段数错误帧会丢弃。
