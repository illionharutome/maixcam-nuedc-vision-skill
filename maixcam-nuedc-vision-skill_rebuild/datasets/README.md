# Dataset workspace

Do not mix unrelated data in this directory. Create a separate scene folder for every task and record lighting, distance, background, camera settings, and task type for every image batch.

Keep raw captures in `raw/`, annotations and metadata in `labeled/`, and generated YOLO layouts in `yolo/`. Keep training, validation, and final test images separate; never evaluate on training images.

No dataset is bundled in the current 2025E phase.
