# 视觉到机械臂主控的后续接口

MaixCAM 未来可提供目标类别、框、角度、置信度、抓取点、抓取方向、距离估计和状态。它不得输出舵机 PWM、绝对角度序列或抓取动作。

MSPM0G3507 或独立机械臂主控必须完成运动规划、限幅、安全检查、目标丢失、超时、停止、回中和执行确认。只有 MCU 状态机确认后才可能允许动作。

当前 `ArmTaskResult` 和 MCU 文件仅占位，初始化保持 `manual_stop=1`、`execution_enabled=0`。不实现机械臂控制，也不实现 ARM 解析。

未来新增 ARM 帧时必须保持 AIM 帧不变，并增加端到端解析与安全状态测试。
