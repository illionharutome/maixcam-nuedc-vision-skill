# 当前测试安全边界

本阶段只验证摄像头、传统视觉、图像误差和文本串口协议，不使用真实激光，不连接任何执行机构。

允许的替代目标：红色 LED、红点图片、手机或电脑屏幕红点、红色圆点纸、红笔点，以及其他低风险替代光点。

禁止事项：

1. 默认启用 405nm 或将其作为当前测试入口。
2. 建议或使用高功率激光。
3. 对人、动物、车辆、窗户、镜面或其他反射面测试。
4. 连接真实激光开关控制。
5. 在视觉与 UART 验证前连接云台、舵机、机械臂或小车执行机构。
6. 让 MaixCAM 直接控制任何执行机构。

`blue_violet_405` profile 必须保持 `enabled: false`。只有用户在现场实测 MaixCAM-Pro 的 LAB 响应、确认光路与安全条件后，才可在未来阶段手动启用；本仓库不自动启用它。
